La practica consistira en poner a prueba el depurador de eclipse en alguno de los programas.java que tenemos 
Para ello adjuntamos una serie de screenshots que estan en la carpeta de DOCUMENTOS_ADICIONALES.

Para poder poner a prueba el debug utilizaremos una clase sencilla.
Para ello rehutilizaremos una de las clases de java utilizada en la asignatura de programacion.
y que mas sencillo que una suma.
En las screenshots aparecen las diferentes modificaciones que se han echo con las variables que se introducen,
marcando el breakpoint viendo como salta el debug a la perspectiva de depuración, y como avanza linea a linea
y accede a las diferentes clases mediante F5
